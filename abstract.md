## ABSTRACT

The rapid proliferation of Internet of Things (IoT) technologies has significantly advanced home automation; however, indoor cultivation methodologies often remain reliant on qualitative intuition rather than quantitative metrics. This project presents the design and implementation of a comprehensive Smart Plant Monitoring System aimed at establishing a data-driven, interactive ecosystem for domestic botany. Through the deployment of precision edge sensors, the system continuously aggregates critical environmental telemetry—specifically soil moisture saturation, ambient temperature, and relative humidity. These physiological parameters are synchronized with a persistent backend architecture and visualized via a dynamic React.js web application, providing users with robust tracking capabilities for both immediate localized conditions and longitudinal environmental trends.

To surpass the limitations of conventional binary-threshold alerts, the system integrates advanced Artificial Intelligence architecture utilizing the Sarvam Large Language Model (LLM). This AI layer actively ingests real-time sensor metrics into its computational prompts, synthesizing environmental data to generate highly contextual, anthropomorphic diagnostic feedback. This effectively enables the monitored plant to interact intelligently with the user. Concurrently, to mitigate the inherent vulnerabilities of cloud-dependent infrastructures—such as API deprecation or network latency—the system is fortified with a deterministic offline algorithmic fallback. This localized edge-computed rule engine evaluates distinct environmental thresholds to guarantee uninterrupted, accurate diagnostic reporting irrespective of external network integrity.

A critical functional enhancement distinguishing this prototype is the deep architectural integration of Web Speech APIs, enabling bidirectional Text-to-Speech (TTS) and Speech-to-Text (STT) capabilities. This auditory processing framework fundamentally transforms the user experience, transitioning the system from a passive graphical interface into an active, screen-independent diagnostic companion. Users can leverage natural language voice queries to initiate hands-free system interactions, prompting the platform to natively synthesize audible, data-driven responses and critical ambient alerts—such as executing vocal warnings when soil saturation surpasses optimal parameters.

Ultimately, this multidisciplinary approach seamlessly translates underlying IoT diagnostics into an accessible, interactive digital entity. By systematically amalgamating precise biometric telemetry, resilient edge-computed fallbacks, and intuitive voice-activated interfaces, the proposed system establishes a progressive paradigm in empathetic human-computer interaction while significantly optimizing autonomous botanical maintenance.
